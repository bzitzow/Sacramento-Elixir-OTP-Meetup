### Installation

OS X:
```console
$ brew update && brew install exercism
```

## Elixir Koans
https://github.com/elixirkoans/elixir-koans.git

Clone the repository and launch the koans.

```console
$ git clone https://github.com/elixirkoans/elixir-koans.git
$ cd elixir-koans/
$ mix deps.get
$ mix meditate
```
